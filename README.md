# Distill

Most learning tools help you collect. Distill helps you *understand*.

Define a topic. Add sources in any format — articles, videos, notes, PDFs, 
raw thoughts. Each time you add new material, Distill synthesizes everything 
into a consolidated body of knowledge and produces a structured update note 
describing exactly what changed, what was added, and what was refined.

Over time, you build a versioned, evolving understanding of any subject — 
the way a codebase evolves through commits, but applied to how you think.
